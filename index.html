Blackstone


Blackstone is a programming language designed for the MCDiamondFire minecraft server.

Hello world example: 
event player.join {
    player.sendMessage("Hello world!");
    plot.setBlock(loc(10, 20, 30), item(diamond_block));
}
You can compile this code using the Shulker compiler. It will be processed through Recode onto DiamondFire.
